關 於 我

ABOUT

Hello! 我是雷富堡，擁有5年的電商美編設計經驗。
雖然非設計相關科系出身，但憑自身對設計的熱情及敏銳度，從設計零經驗到自學設計軟體已累計許多Banner / DM / EDM / 海報 / LOGO / 包裝設計等平面作品。

在上一家就職的電商公司也經歷規劃過2個美妝/保養電商RWD網站、APP的UI設計，自此引起了我對UI設計工作高度的興趣。

離開公司後，開始自行進修 網頁設計 & UI介面設計，並持續透過線上課程與網路資源來提升設計技能及知識。

EXPERIENCE

2018.05 ~   SOHO接案、學習進修

暫離職場半年多來開始嘗試接案，其中比較相關的案子是一家女裝電商 Girl's Monday。
依照業主的需求，幫他們重新規劃成了 RWD 版型，增加影音介面功能，並優化了結帳流程。
Prototype *非最新版本

UI介面設計

至今都是使用PS繪製 Wireframe 及 Prototype，直到知道有 Adobe XD 這套專門為開發原型所設計的軟體後，看了一眼就決定要改用 XD 來設計，不管是 APP 還是 Web Design 都能使用，我相信在設計上能更順手。

網頁設計

除此之外，為了精進自己的設計技能，選擇購買線上課程開始學習網頁設計，從最基礎的 HTML / CSS 開始，再導入 JQuery / RWD，更進一步學習使用Bootstrap套件。

2013.05 ~ 2018.05   明洞國際 | 美編主任

主要負責網站一切視覺設計、系統維護、社群媒體視覺素材以及分配美編組員工作事項。

從公司最初的Yahoo網拍時期，每週的工作內容僅是準備好下週要上檔的商品照、內頁 ; 一直經歷到轉型為保養品公司，並架設了獨立品牌購物網站，隨著商品種類的增加，社群媒體平台的興起，實體店面的成立，因而接觸到不同於以往的設計項目，也累積了 產品視覺素材、店面輔銷物 及 各家社群平台行銷素材 等相關設計經驗。

在職期間這幾年，因為智慧型手機的興起，消費模式的轉變，讓我了解到現今電商不僅是要有吸引眼球的視覺，更要具有直覺性的操作行為、更具流暢的購物動線，進而順利地引導客戶進行轉單。

公司的網站頁面架構主要都是我設計的，直到開始規劃 APP 的 UI 時才開始學習到相關的知識及規範，也同時加深了我對 UI 設計的興趣。

2012.10 ~ 2013.04   名統百貨 | 總務專員

主要負責 採購、固定資產、外包廠商 業務。
退伍後第一份工作，在這半年期間發現這領域的工作並不是我想要的，也想通自己想多了解網拍行業。

離開名統後，利用一個月的時間進修繪圖軟體，也順利地找到網拍美編的工作。
